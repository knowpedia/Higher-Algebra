# [Higher-Algebra](https://knowpedia.github.io/Higher-Algebra)
高等代数，包括行列式和矩阵、线性方程组、线性空间和线性变换、欧几里得空间、双线性函数与辛空间等代数基础知识。

## Issues
使用的时候遇到任何问题或有好的建议，请点击进入[issue](https://github.com/knowpedia/Higher-Algebra/issues)，欢迎参与维护！

## How to use?

开发的时候运行下列命令：

```
npm run dev
```

开发完毕需要发布的时候，请运行下列命令：

```
npm run build
```
